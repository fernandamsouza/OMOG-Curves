# OMOG-Curves

Para executar:

```
	1. python3 omog.py;
	4. Selecione 5 pontos de controle para geração da NURBS;
	2. Após, selecione 4 pontos de controle no plano para geração da Bézier;
	3. A curva Bezier Grau 3 é gerada;
	5. A curva NURBS Grau 4 é gerada com continuidade C0, G1/G2 a Bezier Grau 3.
```
